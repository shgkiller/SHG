微信小程序-跑马灯

 在wxml页面加如下代码：
       
<view class='scroll_view_border'>
<view class="srcoll_view" bindtap="textclick">这武器那都好，就是不吃药只能战斗15秒，15秒内不是你死就是我亡，这才是90级史诗该有的王者风范!</view>

</view>

在wxss页面加如下代码：

.srcoll_view{
position: absolute;
top:0rpx;
height: 100rpx;
font-size: 30rpx;
white-space: nowrap;
line-height: 100rpx;
color: #00bff3;
animation:myfirst 20s linear infinite;
}
@keyframes myfirst
{
  0% {margin-left:750rpx;}
  100% {margin-left:-1500rpx;}
}
.scroll_view_border{
position: relative;
width: 100%;
background: rgba(0, 0, 0, 0.2);
height: 100rpx;
overflow: hidden;
}
